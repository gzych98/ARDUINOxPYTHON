# ARDUINOxPYTHON

Project of Python Tkinter app used to control servomechanism by Arduino Uno with Serial communication.

![alt text](https://github.com/gzych98/ARDUINOxPYTHON/blob/main/img1.png?raw=true)


## Wiring diagram

Components:

- Arduino Uno
- Servo
- DS18B20
- Resistor 4,7kOhm
